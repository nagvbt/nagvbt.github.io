Definition: A pointer is a variable which stores the address of another variable.

2. Explain about pointer Notation ?‚Äã

Consider the declaration,

int i = 3 ;

This declaration tells the C compiler to: (a) Reserve space in memory to hold the integer value. (b) Associate the name i with this memory location. (c) Store the value 3 at this location.

Example:

#include <stdio.h>
int main( )
{
 int i = 3 ;
 
 printf ( "\nAddress of i = %u", &i ) ;
 printf ( "\nValue of i = %d", i ) ;
 printf ( "\nValue of i = %d", *(&i) ) ;

 return 0;
}

8. Explain about Dynamic Memory allocation ?‚Äã

If we need a variable amount of memory that can only be determined during runtime in that case we need some user input to determine the necessary amount of memory space.

malloc, calloc, free, realloc

10. Explain about malloc ?‚Äã

malloc allocates a block of size of memory, returning a pointer to the beginning of the block. Example: char arr; int i; arr = (char) malloc(i+1);

Since malloc() returns a void pointer it is necessary to explicitly typecast it into an appropriate type of pointer.

Example: int p1 = (int)malloc(sizeof(int));

11. Explain about calloc ?‚Äã

Allocate space for array in memory. calloc allocates a block of memory for an array of num elements each of them size bytes long and initializes all its bits to zero. The effective result of an zero-initialized memory block of (num * size) bytes.

void* calloc(size_t num, size_t size);

num - No. of elements to be allocated size - size of elements

NOTE: In calloc and malloc if the function failed to allocate the requested block of memory a NULL pointer is returned.

12. Explain about realloc?‚Äã

voidrealloc(void ptr, size_t size); ptr - pointer to a memory block previously allocated with malloc, calloc or realloc to be reallocated. If this is NULL a new block is allocated and a pointer to it is returned by the function.

  7. What is an ActiveX interface?

  8. What is an ActiveX method?

    A method is an exposed function that can be applied to a control via the IDispatch interface.

  9. What are the properties exposed by ActiveX controls?

    ActiveX controls have four types of properties: l Stock:-> These are standard properties supplied to every control, such as font or color. The developer must activate stock properties, but there is little or no coding involved.

    2 Ambient:-> These are properties of the environment that surrounds the control - properties of the container into which it has been placed. These can't be changed, but the control can use them to adjust its own properties. For example, it can set the control's background color to match the container's background color.

    3 Extended:-> These are properties that the container handles, usually involving size and placement onscreen.

    4 Custom:-> These are properties added by the control developer.
